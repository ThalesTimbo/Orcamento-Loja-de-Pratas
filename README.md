# Orçamento — Loja de Pratas, Joias e Acessórios

Site estático (HTML/CSS/JS) para **orçamento** e **vendas** de uma loja de pratas/joias/acessórios, com:

- **Orçamento pelo WhatsApp** (formulário gera mensagem pronta e abre o WhatsApp)
- **Catálogo / venda virtual** (link configurável)

## Como configurar

Abra o arquivo `index.html` e ajuste estes atributos no `<body>`:

- **WhatsApp da loja**: `data-whatsapp="55DDDNUMERO"`
  - Ex.: `data-whatsapp="5511999999999"`
  - Use **apenas números** (com DDI 55).
- **Link da loja virtual/catálogo**: `data-loja-virtual="https://..."` (ou um link interno)

## Como abrir

Basta abrir o `index.html` no navegador.
